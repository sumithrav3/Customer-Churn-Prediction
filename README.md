# Customer Churn Prediction

## Project Overview
This project predicts whether a customer will leave a company using machine learning classification algorithms.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Models Used
- Logistic Regression
- Random Forest

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix
